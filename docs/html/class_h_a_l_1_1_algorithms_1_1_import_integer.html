Detailed Description

Generic algorithm class that stores an integer value from information in a TTree.

This algorithm imports the information to store an integer value from a specified branch in a TTree. The value from this algorithm is stored in a GenericData object in the UserData under the algorithm's name and <name>:value.

Explaination of the branch map:
The required map is just one that points to the relavent integer value. <name> refers to the name given to this algorithm's constructor.
Required Branch Map:

Integer value
<name>:integer

Example:
In your analysis file, do the following for an integer value:

HAL::Analysis a("sample analysis", "", "truth");
a.AddAlgo(new HAL::Algorithms::ImportInteger("integer value", "import an integer value"));
//...
a.MapBranch("some_integer_branch", "integer value:integer");
