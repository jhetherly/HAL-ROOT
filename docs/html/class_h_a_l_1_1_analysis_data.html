Detailed Description

Class for storage and sharing of data referenced by strings.

This class aids in passing data between algorithms. The common data store is a TList. This causes problems when algorithms must pass plain old data (POD) between each other as a TList may only hold object deriving from TObject. This class provides a way for POD to be stored and retrieved by strings.
