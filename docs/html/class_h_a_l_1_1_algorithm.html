Detailed Description

Base class for all analysis algorithms.

This class serves as the parent class for all HAL analysis and user-defined algorithms. It contains hooks for all parts of the calling TSelector, i.e. SlaveBegin, Process (through Exec and Clear), Terminate, etc. It also has methods for keeping track of how many objects it creates. Most importantly, this class can pass information to other algorithms through a common data store accessed through string names and a few specilized methods.
