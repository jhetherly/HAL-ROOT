1 /*
2  *
3  * This is a general purpose TTree reading class. This class is designed
4  * to handle the most popular types of data that are stored in TTree's
5  * for physics analysis. In particular it can handle scalars, c-arrays,
6  * std::vectors, 2D c-arrays, and std::vectors of std::vectors of the
7  * following types:
8  *
9  * Bool: bool, Bool_t
10  * Integer: int, Int_t, short, Short_t, long, Long_t, long long, Long64_t, signed char
11  * Counting: unsigned int, UInt_t, unsigned short, UShort_t, unsigned long, ULong_t, unsigned long long, ULong64_t, unsigned char, Byte_t
12  * Decimal: float, Float_t, Float16_t, Real_t, double, Double_t, Double32_t, long double, LongDouble_t
13  * String: char, Char_t, char*, Text_t, TString, std::string
14  * Misc: TObjArray, TClonesArray, TRef, TRefArray
15  *
16  * NOTE: This class can only handle TObjArrays, TClonesArrays, and
17  * TRefArrays stored as scalars, c-arrays, and std::vectors.
18  *
19  * */
